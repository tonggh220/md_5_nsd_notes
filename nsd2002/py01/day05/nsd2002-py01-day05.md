# nsd2002-py01-day05

## 列表

- 属于容器、序列、可变类型

```python
>>> from random import randint
>>> l1 = [randint(1, 100) for i in range(10)]
>>> l1
[21, 65, 58, 1, 92, 27, 27, 11, 59, 60]
>>> len(l1)
10
>>> l1[0] = 59
>>> l1[2:4]
[58, 1]
>>> l1[2:4] = [10, 20, 30]
>>> l1
[59, 65, 10, 20, 30, 92, 27, 27, 11, 59, 60]

```
