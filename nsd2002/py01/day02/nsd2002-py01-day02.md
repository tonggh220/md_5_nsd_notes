## 字典

- 字典是由键-值(key-value)对构成的映射数据类型
- 通过键取值，不支持下标操作
- 字典是没有顺序的

```python
>>> d1 = {"name": "tom", "age": 20}
>>> len(d1)
2
>>> 'tom' in d1   # tom是字典的key吗
False
>>> 'name' in d1  # name是字典的key吗
True
>>> d1['name']    # 通过key取value
'tom'
```

### 数据类型总结

#### 按存储模型分类

- 标量：字符串、数字
- 容器：列表、元组、字典

#### 按访问模型分类

- 直接：数字
- 顺序：字符串、列表、元组
- 映射：字典

#### 按更新模型分类

- 不可变：数字、字符串、元组
- 可变：列表、字典

```python
>>> s1 = 'python'
>>> l1 = [10, 20, 30]
>>> s1[0]
'p'
>>> l1[0]
10
>>> l1[0] = 100
>>> s1[0] = 'P'   # 字符串不可变，不能对它直接修改
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'str' object does not support item assignment
>>> s1 = 'Python'  # 整体重新赋值

>>> l1
[100, 20, 30]
>>> l2 = l1     # l1和l2指向内存的相同地址
>>> l2
[100, 20, 30]
>>> l2[-1] = 300  # 修改l2也会影响到l1
>>> l2
[100, 20, 300]
>>> l1
[100, 20, 300]
```
