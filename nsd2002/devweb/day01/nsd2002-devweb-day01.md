## web开发

- 前端：离用户近的一端是前端。
  - html：超文本标记语言，实现网站的内容
  - css：层叠样式表，也叫级联样式表，它实现网站的表现形式
  - javascript：简称js，实现网站的行为
- 后端：离用户远的一端，也就是服务器端，是后端。
  - python
  - java
  - php
  - asp.net

## html

- 结构

```html
<!DOCTYPE html>     <!--声明文档类型-->
<html lang="en">    <!--html是根标记，所有标记都需要写在html标记内部-->
<head>              <!--头部消息-->
    <meta charset="UTF-8">  <!--元数据-->
    <title>Title</title>    <!--标题-->
</head>
<body>  <!--出现在浏览器窗体中的内容-->

</body>
</html>
```
